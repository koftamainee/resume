\section{Проекты}

\subsection*{\href{https://github.com/koftamainee/search-engine}{Search Engine}}
\hfill \textit{Go, Docker}

Микросервисная поисковая система. Веб-краулер с очередью на Redis и поддержкой robots.txt,
BM25-индексация через Meilisearch, REST API на Go, фронтенд на Nuxt.
Инфраструктура развёртывается через Docker Compose.
Был развёрнут в Yandex Cloud по адресу \href{https://search.koftamainee.ru}{search.koftamainee.ru};
в настоящее время сервис неактивен в связи с исчерпанием пробных средств.

Сейчас в разработке: сервис метрик на Rust для сбора page-quality сигналов
и PageRank на графе ссылок для переранжирования результатов.

\subsection*{\href{https://github.com/koftamainee/mantle}{Mantle}}
\hfill \textit{C++, Vulkan}

Модульный игровой движок.
Vulkan-рендерер с Frame Graph для автоматической синхронизации GPU,
система виртуальной памяти с кастомными аллокаторами (arena, tlsf),
интеграция с flecs для ECS-архитектуры и с Jolt Physics для физической симуляции,
кастомные скрипты для кодогенерации и форматирования, CI/CD с авто-сборкой под Linux и Windows.

Сейчас в разработке: выделение стабильного Engine API для интеграции с game кодом,
написание высокоуровневых абстракций для загрузки ассетов и сцен, а также кастомизция рендеринга из игрового кода.

\subsection*{\href{https://github.com/koftamainee/mint}{Mint} / \href{https://github.com/koftamainee/mintload}{Mintload}}
\hfill \textit{Rust, C}

Ассет-пайплайн для Mantle (каноническое название Mantle INput Tool): конвертация glTF/GLB в бинарные форматы удобные для прямого копирования на GPU,
система сцен через текстовые .mscn файлы и конвертация их в бинарный .mscb формат,
система работы с проектом и инкрементальная сборка.
Mintload --- небольшая библиотека на C для быстрой загрузки файлов в движок через mmap. Также есть wrapper под C++.

Сейчас в разработке: шейдерный компилятор (.mshd), поддержка hot-reload и формат релиз-архива (.mpak).
